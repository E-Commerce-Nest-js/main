## Описание

Интернет магазин одежды.
E-Commerce проект, построенный на микросервисной архитектуре. С использованием фреймворка Nest.js.

## Технологии
 - NGINX
 - Nest.js
 - MongoDB
 - RabbitMQ
 - Swagger
 - ~~Redis~~
